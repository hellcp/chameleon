Contributing to openSUSE Documentation

Contributions to the guides listed above are welcome. You can report issues by clicking on the Report Bug link in the headlines of the HTML versions.

You may also fork the GitHub repository and send us pull requests. For more information, see README. If you are interested in writing larger amounts of text, see our Documentation Styleguide for further help.

To get in touch with the documentation team:

  • Mail us at doc-team@suse.com
  • Join the IRC channel #opensuse-doc on freenode
  • Subscribe to our mailing list opensuse-doc@opensuse.org
